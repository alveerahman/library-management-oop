# Library Management System (OOP) 📚

This is the 3rd project of my **90-Day Software Engineering Sprint**. As a **Final Year CSE Student**, I have implemented this project to master **Object-Oriented Programming (OOP)** principles.

## 🚀 Learning Milestone
In this project, I moved beyond procedural programming. I used **Classes and Objects** to represent real-world entities (Books and Libraries), which is a core skill for any Software Engineer.

## ✨ Features
- **Object Modeling:** Each book is an object with its own attributes.
- **Encapsulation:** Managing book status (Available/Borrowed) within class methods.
- **Dynamic Management:** Add and track books in real-time.
- **Search Logic:** Case-insensitive search to borrow books.

## 🛠️ Technical Skills Applied
- **Classes & Objects:** Used `__init__` and `self` for state management.
- **Methods:** Implementation of logic inside classes.
- **String Representation:** Using `__str__` for clean data display.

## 📖 How to Run
1. Clone the repository.
2. Run `python library_system.py`.
